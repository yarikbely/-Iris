## Нейронная сеть: реализация и классификация датасета Iris

---

### 1. Цель работы

Реализовать искусственный нейрон с сигмоидной функцией активации, градиентный спуск и алгоритм обратного распространения ошибки (backpropagation). На основе реализованного кода решить задачу бинарной классификации датасета Iris двумя способами:
- Одним нейроном (логистическая регрессия)
- Нейронной сетью из 2 скрытых слоев по 10 нейронов

---

### 2. Теоретическая часть

#### 2.1 Искусственный нейрон

Искусственный нейрон — это математическая модель, вдохновленная биологическими нейронами. Он принимает на вход несколько сигналов, умножает их на веса, суммирует с порогом (bias) и пропускает через функцию активации.
