# Лабораторная работа №4: Нейронная сеть: реализация и классификация датасета Iris

## Цель работы
Реализовать искусственный нейрон с сигмоидной функцией активации, градиентный спуск и обратное распространение ошибки. Решить задачу классификации Iris одним нейроном и сетью 2x10 нейронов.

## Реализованные компоненты

### 1. Класс Neuron
- Сигмоидная активация: σ(z) = 1/(1+e^(-z))
- Прямой проход: вычисление взвешенной суммы и активации
- Обратный проход: вычисление градиентов и обновление весов

### 2. Класс Layer
- Группировка нейронов в слой
- Массовый forward/backward

### 3. Класс NeuralNetwork
- Многослойная архитектура
- Backpropagation
- Мини-батчный градиентный спуск
